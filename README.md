# 通讯录管理系统简介 java120

## 项目概述

通讯录管理系统是基于前后端分离模式开发，旨在为用户提供一个高效、便捷的通讯录管理解决方案。

## 技术栈

本项目采用以下技术构建：

- 后端：Spring Boot
- 前端：Vue
- 数据库：MySQL

## 功能模块

### 用户角色

系统包含以下两种角色：

- 管理员
- 其他用户

### 功能列表

1. 注册与登录
   - 用户注册
   - 用户登录
2. 通讯录公告
   - 查看公告
   - 管理公告（仅限管理员）
3. 通讯录用户管理
   - 增加用户
   - 删除用户
   - 修改用户信息
   - 查询用户信息
4. 通讯录组管理
   - 编辑用户组（如小学组、初中组、高中组、大学组、家人组）
5. 通讯录菜单管理
   - 编辑通讯录组对应的菜单功能权限
6. 个人信息管理
   - 修改个人用户信息（如用户头像）

## 系统角色

系统中的角色分为管理员和其他用户，管理员具有管理通讯录公告、用户和菜单权限的能力，其他用户则主要负责维护个人通讯录信息。

## 核心亮点

- 前后端分离，系统架构清晰，易于维护与扩展。
- 角色权限分明，保障系统安全性和稳定性。
